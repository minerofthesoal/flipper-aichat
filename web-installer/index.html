Flipper AI Chat — Devboard Installer

Flashes the ESP32-S2 bridge firmware onto the Flipper Zero WiFi Devboard, straight from this page. Chrome / Edge on desktop only (Web Serial).

1Connect the devboard

Plug the WiFi Devboard into your computer over its own USB port (not through the Flipper). Then click below and pick its serial port when the browser prompts you.

2Configure WiFi

The install flow above walks you straight into WiFi setup after flashing (powered by Improv Wi-Fi). If you ever need to reconfigure WiFi on an already-flashed board later, come back to this page, click the button above again, and choose "Configure Wi-Fi" without reflashing — or send REPROVISION from the Linux TUI's Serial mode first if the board is already running.

3Point it at LM Studio

Improv only carries WiFi credentials, so the LM Studio server address is set separately, over the same serial port. Enter your PC's address (LAN IP at home, e.g. http://192.168.1.42:1234, or a Tailscale address — see docs/SETUP.md) and send it below.

No Web Serial support, or prefer a terminal? The same thing works from ai-chat-tui's Serial mode, or any serial monitor at 115200 baud: send SETSERVER|http://your-pc:1234.
